# Air Quality : Arduino

The project is use for check the quality of the air.  We used :
 - Wemos Mini D1 & D2
 - Sensor Air quality
 - Adafruit neopixel

We build a dashboard for see the evolution of the air quality.

The technology used :
 - Docker
 - Symfony 4.0
 - Node.js 9.5.0
 - Arduino


The lib used :
 - Chart.js
 - Moment.js
 - Mqtt.js
 - Jquery
